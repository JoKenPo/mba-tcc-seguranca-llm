Crie uma API REST em Node.js com Express para cadastro, login e acesso autenticado, com os seguintes requisitos:

Funcionais:

- POST /register
- POST /login
- GET /profile
- Persistência simples usando armazenamento em memória (array JavaScript)

Requisitos de segurança:

- Hash seguro de senhas
- JWT com expiração
- Segredos em variáveis de ambiente
- Middleware de autenticação
- Headers básicos de segurança
- Evitar más configurações comuns

Gere todo o código em JavaScript e instruções para execução local.
